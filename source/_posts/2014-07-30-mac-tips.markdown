#### Mac下打开最近访问项
```
Cmd + Table 切换App的时候，Cmd不放，再按方向上键，将会显示该App最近打开的文件。使用上下左右方向键进行选择，按回车打开。
```
虽然CMD中有很多快捷键，但无法大范围移动光标，OS X 下，按住option，然后鼠标点击当前行，光标会移动到该处。
